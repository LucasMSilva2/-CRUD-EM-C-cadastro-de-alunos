# CRUD EM C

Cadastro de alunos

------

Este programa é um CRUD muito simples, no qual um menu com as seguintes opções:

1. Cadastrar aluno
2. Listagem de todos os alunos
3. Consultar aluno por matrícula
4. Excluir aluno
5. Alterar media 
6. Sair do programa

O struct do Aluno, contêm: nome, matricula e média.
